# PuzzlePro v14.0 (旗舰增强版)

一个基于纯前端技术（Vue 3 + Canvas + Tailwind CSS）实现的高性能图片处理与拼图工具。该项目完全在本地浏览器中运行，无需任何后端服务器支持，具备极高的安全性和响应速度。

---

## 🚀 核心功能特性

### 1. 多渠道图片导入与便捷管理
* **追加与多选**：支持点击按钮一次性选择多张本地图片，并支持后续持续追加图片。
* **全局全局粘贴（Ctrl+V）**：完美支持 PC 端用户从系统剪贴板中直接粘贴图片，大幅优化操作流。
* **列表轻量化管理**：提供直观的缩略图列表，支持一键清空、单张删除或快捷设置主图。

### 2. 三大图像处理模式
* **经典拼图 (Grid)**：自动将多张图片组合为网格布局。提供“竖长”、“正方”、“横宽”三种版面倾向切换，并支持“裁剪铺满 (Cover)”与“完整保留 (Contain)”两种填充算法。
* **画中画 (PiP 贴纸)**：以首张图片为画布背景，其余图片转换为可独立交互的浮动贴纸。用户可单独（或同步）调节每张贴纸的**缩放比例、圆角大小和阴影深度**。
* **批量水印 (Batch)**：专为批量防盗图设计的快捷工具。可将设计好的水印无缝套用到所有上传的独立图片中，并提供快捷的前后翻页预览功能。

### 3. 专业级双模式水印系统
支持文本换行输入，并提供两种渲染维度：
* **贴纸模式 (Sticker)**：在画布中生成独立的水印文本块，内置黄油体、龙苍体、手写体等 5 种中文字体，支持自定义颜色、旋转角度和字号大小。**支持在画布中直接通过鼠标或手指对水印进行拖拽定位、旋转句柄操控及无缝缩放**。
* **平铺模式 (Tile)**：全荧幕盲打防盗水印，支持高自由度调节平铺间距、水印字号以及不透明度。

### 4. 高级导出与共享
* **高质量保存**：一键导出并下载为高清晰度的 `.jpg` 格式图片。
* **免下载复制**：集成现代剪贴板 API，直接将生成的图片转换为二进制数据流写入系统剪贴板，支持在微信、钉钉等聊天软件中直接 `Ctrl + V` 发送。
* **队列批量下载**：在批量水印模式下，程序会自动按队列依次渲染并依次触发所有图片的下载任务。

## 🛠️ 技术实现亮点

### 1. 数据驱动与实时预览（Vue 3 + Canvas）
项目核心通过 Vue 3 的 `watch` 深度监听响应式配置对象（`config`）与图片数据集（`images`）。当用户调整任何滑块或修改文本时，系统会自动触发画布重绘函数 `draw()`，实现毫秒级的**实时预览**效果。

### 2. 超清分辨率渲染架构
为了保证导出图片的印刷级清晰度，Canvas 的内部渲染基准分辨率（MAX）被锁定为 **3000 像素**，而前端 UI 界面通过 Tailwind CSS 的弹性布局进行自适应物理缩放显示，完美解决了常规前端工具导出图模糊的痛点。

### 3. 逆向矩阵碰撞检测算法
在贴纸水印处于旋转状态时，传统的正交碰撞检测（判断点击坐标是否在 X/Y 范围内）会失效。本项目在鼠标事件处理器中引入了**逆向坐标变换**。
当触发点击时，系统会将鼠标的相对偏移坐标 $(dx, dy)$ 结合当前水印的旋转角度 $\theta$ 进行反向旋转，将其投影回未旋转的正交空间：

$$localX = dx \cdot \cos(-\theta) - dy \cdot \sin(-\theta)$$
$$localY = dx \cdot \sin(-\theta) + dy \cdot \cos(-\theta)$$

利用变换后的 `localX` 和 `localY`，程序可以精确判断用户是否点中了旋转后的文本主体、顶部的旋转拉杆或右下角的等比缩放滑块，实现了丝滑的图形级交互交互。

### 4. 异步队列与原生 Web API 深度整合
* 充分利用 `FileReader API` 将用户输入的 File 转化为 Base64，提升本地加载效率。
* 使用 `canvas.toBlob()` 与 `navigator.clipboard.write()` 异步组合，安全高效地实现跨应用的图片粘贴共享。
* 批量下载阶段引入异步延迟定时器 `await new Promise(r => setTimeout(r, 200))`，为浏览器留出充足的重绘缓冲时间，有效防止高频触发下载导致的浏览器崩溃或任务丢失。

## 📦 项目部署与使用

由于本项目采用**单文件纯前端架构**，部署极其简单：

1. 克隆或下载本仓库中的 `PuzzlePro v14.html` 文件。
2. 双击该 HTML 文件，即可在任何现代浏览器（Chrome、Edge、Safari 等）中完美运行。
3. 也可以直接将其托付于 **GitHub Pages**、**Vercel** 或 **Netlify** 等静态文件托管平台，一键生成在线工具网站。
